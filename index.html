Sobre o Projeto

Este aplicativo conversor foi desenvolvido como parte da disciplina Conceitos da Computação, com o objetivo de aplicar na prática conceitos fundamentais como lógica computacional e representação de dados.

O sistema realiza conversões entre diferentes bases numéricas utilizadas na computação, como decimal, binário e hexadecimal, permitindo compreender como os dados são representados internamente nos computadores.

Os computadores trabalham exclusivamente com o sistema binário (0 e 1). Já o sistema hexadecimal é amplamente utilizado para representar grandes valores de forma mais compacta.

Exemplos:

  • 10 → 1010 (binário)
  • 1101 → 13 (decimal)
  • 255 → FF (hexadecimal)

O projeto foi desenvolvido de forma totalmente autônoma, desde a lógica de conversão até a implementação, representando um marco importante no aprendizado em programação.
